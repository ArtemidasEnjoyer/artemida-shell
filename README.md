  .config/quickshell 
directory

and install quickshell

for Arch:
  `yay -S quickshell-git`
  `paru -S quickshell-git`

 - `hyprland` - window manager
 - `matugen` -  for colors (i know about built in color function in quickshell, but i prefer to use matugen)
